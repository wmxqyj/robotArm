## 1. 产品概述

### 1.1 产品简介

RobotArm-6DOF是一款高性能6自由度机械臂，采用STM32F407高性能微控制器，结合闭环步进电机和先进的运动学算法，实现精确的位置控制和轨迹规划。

### 1.2 主要特点

- **6自由度设计**：灵活的工作空间，可到达复杂姿态
- **闭环控制**：采用闭环步进电机，无丢步，高精度定位
- **实时运动学解算**：内置正/逆运动学算法，支持笛卡尔空间控制
- **多种控制方式**：支持串口命令、Python上位机、手柄实时控制
- **轨迹规划**：支持直线插值、时间函数轨迹
- **FreeRTOS实时系统**：多任务并行处理，响应及时

### 1.3 应用场景

- 教育科研：机器人学教学、运动学算法验证
- 实验研究：运动控制算法开发
- 自动化演示：拾取放置、轨迹跟踪
- 创客项目：智能机器人开发

## 4. 功能特性

### 4.1 运动学功能

#### 正运动学（Forward Kinematics）
- 功能：根据关节角度计算末端位姿
- 输入：6个关节角度 [θ1, θ2, θ3, θ4, θ5, θ6]
- 输出：末端位置 [x, y, z] 和姿态 [roll, pitch, yaw]
- 算法：DH参数法

#### 逆运动学（Inverse Kinematics）
- 功能：根据目标位姿计算关节角度
- 输入：目标位置 [x, y, z] 和姿态 [roll, pitch, yaw]
- 输出：6个关节角度
- 特点：多解选择，选择最接近当前姿态的解

### 4.2 控制功能

| 功能 | 说明 |
|------|------|
| 单关节控制 | 相对旋转、绝对旋转 |
| 多关节同步控制 | 6关节同时运动到目标角度 |
| 笛卡尔空间控制 | 直接控制末端位置和姿态 |
| 轨迹规划 | 直线插值、时间函数轨迹 |
| 远程控制 | 手柄实时控制末端运动 |

### 4.3 保护功能

| 功能 | 说明 |
|------|------|
| 软件限位 | 关节角度范围限制 |
| 硬件限位 | 限位开关保护 |
| 堵转保护 | 电机堵转检测与自动保护 |
| 急停功能 | 立即停止所有运动 |

### 4.4 舵机功能

MG996R舵机控制功能：
- 角度设置：0° ~ 180°
- 平滑运动：指定运动时间
- 速度控制：指定运动速度
- 实时读取：获取当前角度

## 5. 快速入门

### 5.1 硬件连接

1. **电源连接**
   - 连接24V开关电源到主控板电源接口
   - 确认电源正负极正确

2. **电机连接**
   - 连接6个Emm_V5驱动器到CAN总线
   - 驱动器地址设置：1, 2, 3, 4, 5, 6

3. **舵机连接**
   - MG996R舵机信号线连接到TIM3 CH1
   - 舵机电源独立供电（6V）

4. **通信连接**
   - USB转串口线连接USART1（PA9-TX, PA10-RX）
   - 连接电脑USB口

### 5.2 软件准备

1. **安装Python环境**（可选，用于上位机）
   ```bash
   pip install pyserial pygame
   ```

2. **确认串口号**
   - Windows: 设备管理器查看COM口号
   - Linux: `/dev/ttyUSB0` 或 `/dev/ttyACM0`

## 7. 例程说明

### 7.1 Python上位机例程

#### robot_arm.py - 完整控制API

**功能**：提供完整的机械臂控制Python类

**主要方法**：
```python
from robot_arm import RobotArm

# 创建连接
robot = RobotArm(port="COM3", baudrate=115200)
robot.connect()

# 获取关节角度
joints = robot.get_joints()
print(f"当前关节角度: {joints}")

# 设置关节角度
robot.set_joints([0, 45, 30, 0, 90, 0], wait=True)

# 获取末端位姿
pose = robot.get_eepose()
print(f"末端位姿: {pose}")

# 正运动学
result = robot.forward_kinematics([0, 45, 30, 0, 90, 0])

# 逆运动学
target_pose = {'x': 200, 'y': 0, 'z': 150, 'roll': 0, 'pitch': 45, 'yaw': 90}
joints = robot.inverse_kinematics(target_pose)

# 断开连接
robot.disconnect()
```

#### robot_joystick.py - 手柄实时控制

**功能**：使用joycon手柄实时控制机械臂

**使用方法**：
```bash
python robot_joystick.py -p COM3 -b 115200
```

**手柄映射**：
- 左摇杆：控制X、Y方向运动
- 右摇杆：控制旋转
- 扳机：控制Z方向运动
- Q键：退出

## 9. 常见问题

### Q1: 串口连接失败怎么办？

**A**: 
1. 检查USB转串口驱动是否安装
2. 确认串口号是否正确（设备管理器查看）
3. 检查波特率是否为115200
4. 尝试重新插拔USB线

### Q2: 关节不动或运动异常？

**A**:
1. 检查电机电源是否连接
2. 确认CAN通信是否正常
3. 检查驱动器地址设置（1-6）
4. 查看是否有堵转保护触发

### Q3: 逆运动学求解失败？

**A**:
1. 检查目标位姿是否在工作空间内
2. 确认姿态角是否合理
3. 尝试调整目标位置
4. 检查关节限位设置

### Q4: 舵机不响应？

**A**:
1. 检查舵机电源（需要6V供电）
2. 确认PWM信号线连接
3. 测试舵机是否损坏

### Q5: 如何校准零点？

**A**:
```
# 方法1：设置当前位置为零点
abs_zero

# 方法2：执行回零操作
abs_restore
```

## 10. 安全注意事项

### 10.1 使用安全

⚠️ **警告**

1. **通电前检查**：确认电源电压和极性正确
2. **工作空间**：确保机械臂运动范围内无人员和障碍物
3. **负载限制**：不要超过最大负载能力
4. **急停准备**：熟悉急停操作方法

### 10.2 电气安全

1. 使用合格的电源和线缆
2. 确保接地良好
3. 避免在潮湿环境使用
4. 定期检查线缆连接

### 10.3 机械安全

1. 定期检查紧固件
2. 注意关节限位
3. 避免碰撞和冲击
4. 长时间不用时断电

### 10.4 软件安全

1. 不要在运动中突然断电
2. 使用软件限位保护
3. 测试新程序时降低速度
4. 保存重要参数配置
